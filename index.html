Instructions

1. Click on the button below "Unconcious Realities" to go the the project's page.

2. The program will wait for your click button to start but the timer will start immedietly.

3. When the result is shown, a small task bar will appear and you can choose to see some information about the result, save the image or restart it and do it again.

4. The program needs a window or a device with at least 1024x576 pixels to work
